Inventory Management System (SQL Project)
Project Overview
A simple SQL-based Inventory Management System that manages products, suppliers, customers, orders, and stock levels.
The project includes database schema design, relationships, foreign keys, and triggers to automatically update inventory.

Skills & Technologies Used
MySQL, SQL Queries
DDL, DML, DQL
Primary & Foreign Keys
Joins, Group By, Aggregations
Triggers

Database Tables
Products – Product details & stock
Suppliers – Supplier information
Customers – Customer records
Orders – Customer orders
Order_Details – Items in each order
Inventory_Transactions – Stock IN/OUT logs
